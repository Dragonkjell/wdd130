Overview

Purpose

I would like to create a place to dice/coin pouches that I make. I've used use Facebook marketplace to advertise and sell, but I think it would be so much easier and more efficient to have one spot that will show people exactly what I make, as well as list prices. The site would also have my contact info available. After the site is made, I could attach a QR code for people to easily find it.

Audience

The main target demographic is tabletop gamers.
